#  Curso de Práctica de Redes y Hacking Ético

##  Descripción del Proyecto
Este curso y repositorio de documentación está **basado fundamentalmente para practicar Redes**, comprender la infraestructura subyacente (TCP/IP, Modelo OSI) y aplicar estos conocimientos en entornos de Ciberseguridad.

##  Objetivos
* Dominar los protocolos de comunicación (TCP, UDP, IP, ARP).
* Entender el encapsulamiento de datos y análisis de paquetes.
* Preparar las bases técnicas para certificaciones de seguridad ofensiva.
